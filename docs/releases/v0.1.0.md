# Hermes Agent Console v0.1.0

首个可分发的 Hermes Agent 桌面 GUI / sidecar 控制台版本。  
_The first distributable desktop GUI and sidecar console release for Hermes Agent._

## 本版重点

- 中文优先桌面界面，内置 English 切换
- Electron GUI 正式托管真实 Hermes sidecar 生命周期
- 本地启动 Hermes Dashboard，并在桌面内嵌展示
- 本地启动 Hermes Gateway API，提供 `/health`、`/v1/runs` 与 SSE 事件流
- 新增桌面内的 TUI / CLI 控制桥，可直接拉起真实 Hermes 终端命令
- 支持运行时启停、日志查看、打开 Hermes Home、打开日志目录与开源说明
- GitHub Releases 提供 Windows、macOS、Linux 多平台产物

## 实际包含的组件

| 组件 | 内容 |
| --- | --- |
| 桌面壳 | Electron 主进程、preload、React GUI |
| sidecar | 由本地 Python 启动的 Hermes Dashboard 与 Gateway |
| 上游来源 | `vendor/hermes-agent` vendored runtime 与 Web 资源 |
| 数据目录 | Hermes Home、日志、附件、导出、更新目录 |
| 发布资产 | Windows `setup.exe`/`portable.exe`，macOS `dmg`/`zip`，Linux `AppImage`/`tar.gz` |

## 适合谁

- 想直接安装和使用 Hermes Agent 桌面版的用户
- 想在本地观察 Hermes Dashboard 与运行日志的开发者
- 需要把 Hermes runtime 做成真实桌面产品的开源维护者

## 验证记录

发布前执行的核心验证：

- `npm run build`
- `npm run smoke`
- `npm run test:e2e`
- Windows 打包与安装校验

GitHub tag 发布流程会继续构建并上传：

- Windows：`setup.exe`、`portable.exe`
- macOS：`dmg`、`zip`
- Linux：`AppImage`、`tar.gz`

## 已知边界

- `v0.1.0` 重点是把 GUI、sidecar 生命周期、本地日志和分发链路做完整
- 更深入的更新体验、权限边界和桌面产品打磨会在后续版本继续推进

## 版本定位

`v0.1.0` 的意义不是“做一个 Hermes 外观壳”，而是第一次把真实 Hermes Agent sidecar 以桌面 GUI 的方式稳定交付出来。
