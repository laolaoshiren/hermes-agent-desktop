# Hermes Agent Desktop v0.1.0

## Highlights

- Chinese-first desktop AI workspace with optional English UI
- Real streaming chat through the local desktop adapter
- Provider support for OpenAI, OpenRouter, Anthropic, Ollama, and OpenAI-compatible endpoints
- Local sessions, attachment handling, diagnostics export, and health/status pages
- Open-source project structure with the upstream `hermes-agent` source vendored under `vendor/hermes-agent`

## Packaging

- Windows: `setup.exe` and `portable.exe`
- macOS: `dmg` and `zip` via GitHub Actions release workflow
- Linux: `AppImage` and `tar.gz` via GitHub Actions release workflow

## Notes

- This release is usable in direct-provider mode on Windows today
- Hermes-native runtime bridging remains a follow-up step because upstream Hermes currently expects a stronger Python / WSL environment on Windows
- The desktop product architecture already keeps runtime-manager and adapter boundaries in place for that next integration phase

## Validation

- `npm run build`
- `npm run smoke`
- `npm run dist:win`
